## Support

### Questions / help

- Use **GitHub Discussions** (recommended) for “how do I…” questions and general support.
- Use **GitHub Issues** for actionable bugs and feature requests.

### Bugs

When filing a bug, include:

- What you expected vs what happened
- Minimal reproduction steps
- Logs/tracebacks (redacting secrets)
- OS + Python version

### Security

For security issues, please see `SECURITY.md` and do not post details publicly.
